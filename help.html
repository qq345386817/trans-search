Quick guide: setup and everyday use

Most controls live in the extension popup, and auto-translate is ready only on supported search pages.

Get started fast

  1. Install Translate Before Search and enable it in Safari.
  2. Open a supported search page.
  3. Choose your target language in the popup.
  4. Leave auto-translate on if you want it ready by default, or switch the site to manual-only when you want more control.

Core controls

  • Target language: choose which language the query should be translated into.
  • Open results in: current tab or new tab.
  • Skip same language: avoid translating when the query already looks like the target language.
  • Ignored languages: keep auto-translate from running on selected query languages.
  • Site mode: keep a site in manual-only mode if you do not want automatic translation there.

What to do next if it does not work

  • If nothing happens, first confirm you are on a supported search page.
  • If the popup says manual-only, turn auto-translate back on globally or switch the current site back to Follow global.
  • If a query should not translate automatically, add that language to Ignored languages or set the site to manual-only.
  • If translation fails temporarily, try manual translate once, then retry from the same supported search page.

Need more help?

The popup also includes a short “What to do next” hint that changes with the current page state.
