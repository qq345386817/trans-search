Help

Setup and everyday use

Most features are available directly from the extension popup on supported search pages.

Get started

  1. Install Translate Before Search and enable it in Safari.
  2. Open a supported search engine results page.
  3. Choose your target language in the extension popup.
  4. Leave auto-translate on, or translate manually when needed.

Useful settings

  • Target language: choose which language the query should be translated into.
  • Open results in: current tab or new tab.
  • Skip same language: avoid translating when the query already looks like the target language.
  • Ignored languages: keep auto-translate from running on selected query languages.
  • Site mode: keep a site in manual-only mode if you do not want automatic translation there.

Troubleshooting

  • If nothing happens, confirm you are on a supported search results page.
  • If you prefer less automation, turn off auto-translate and use manual translation only.
  • If a query should not translate automatically, add that language to ignored languages or set the site to manual-only mode.
