Search across languages, without breaking your flow.

Translate Before Search translates your query before results open, so it is easier to reach information that may be clearer, newer, or more complete in another language.

Open help Learn how it works

Why this exists

Useful information is not distributed evenly across the web. Some topics are better covered in different languages, and searching in only one language can leave good results behind.

Translate before results open

Skip the usual copy, switch, and paste routine. On supported search pages, the extension can translate your query automatically before the destination results load.

Keep manual control

If you prefer a lighter touch, you can translate manually, keep certain sites in manual-only mode, and still use the extension only when it helps.

Adjust it to your workflow

Choose where results open, switch between recent target languages, skip same-language searches, and ignore selected query languages when needed.

Supported search pages

Translate Before Search supports major search engines including Google, Bing, Baidu, and Sogou, plus other supported search result pages as the extension evolves.
